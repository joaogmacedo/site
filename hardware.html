O que é uma CPU?

A CPU (Unidade Central de Processamento) é o cérebro do computador, responsável por executar instruções e processar dados, realizando cálculos e operações lógicas para que o sistema funcione.

O que é um Armazenamento?

O armazenamento é o componente que guarda dados e informações no computador, podendo ser temporário ou permanente. Exemplos incluem discos rígidos (HDD), unidades de estado sólido (SSD) e memória flash.

O que é uma Memória?

A memória, geralmente referida como RAM (Memória de Acesso Aleatório), é onde o computador armazena temporariamente dados e programas que estão em uso, permitindo acesso rápido para a CPU.

O que é uma Fonte de alimentação?

A fonte de alimentação é o componente que converte a energia elétrica da tomada em voltagens adequadas para alimentar todos os componentes do computador.

O que é um Teclado?

O teclado é um dispositivo de entrada usado para digitar textos e comandos no computador.

O que é um Monitor de Vídeo?

O monitor é o dispositivo de saída que exibe imagens, vídeos e informações visuais ao usuário.

O que é um Mouse?

O mouse é um dispositivo de entrada que permite ao usuário interagir com a interface gráfica do computador, controlando o cursor na tela.

O que é uma Impressora?

A impressora é um periférico que produz uma cópia física (impressa) de documentos e imagens digitais.

O que é um Plotter?

O plotter é uma impressora especializada usada para produzir desenhos técnicos, gráficos e grandes formatos, geralmente em áreas como engenharia e arquitetura.

O que é um Scanner?

O scanner é um dispositivo que digitaliza documentos ou imagens físicas, transformando-os em arquivos digitais para o computador.

O que é um Gabinete?

O gabinete é a estrutura que abriga e protege todos os componentes internos do computador, como a placa-mãe, CPU, memória e fonte de alimentação.

O que é uma Web Cam?

A webcam é uma câmera digital conectada ao computador, usada para capturar vídeo ao vivo, como em videochamadas e gravações.

O que são Caixas de Som?

As caixas de som são dispositivos de saída que reproduzem áudio, permitindo ouvir sons, músicas, vídeos e outros conteúdos sonoros.

O que é um Microfone?

O microfone é um dispositivo de entrada que capta sons e os converte em sinais digitais para serem processados pelo computador.

O que é um Head Set?

O headset é um conjunto de fones de ouvido com microfone embutido, usado para ouvir áudio e falar, muito comum em jogos, chamadas e gravações.

O que são Interfaces sem Fio?

Interfaces sem fio permitem a comunicação e conexão entre dispositivos sem o uso de cabos, utilizando tecnologias como Wi-Fi, Bluetooth e NFC.

O que é um Áudio?

Em informática, áudio refere-se à entrada, processamento e saída de sons, incluindo dispositivos como microfones, alto-falantes e placas de som.

O que é uma Rede (Ethernet)?

A rede Ethernet é um padrão de conexão cabeada para interligar computadores e dispositivos em uma rede local (LAN), permitindo a troca de dados rápida e segura.

O que é um VGA?

VGA (Video Graphics Array) é uma interface de vídeo que transmite sinais analógicos do computador para o monitor. É uma conexão mais antiga e menos usada atualmente.

O que é um HDMI?

HDMI (High-Definition Multimedia Interface) é uma interface digital usada para transmitir áudio e vídeo em alta definição entre dispositivos, como computadores e monitores ou TVs.

O que é um USB?

USB (Universal Serial Bus) é uma interface padrão para conectar dispositivos ao computador, usada para transferência de dados e fornecimento de energia.
